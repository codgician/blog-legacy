题面

对于序列 A={a1,a2,…,an}A = { a_1, a_2, \dots, a_n }A={a1​,a2​,…,an​}，定义 RMQ(A,l,r)\mathbf{RMQ}(A, l, r)RMQ(A,l,r) 为原序列区间 [l,r][l, r][l,r] 内最大值的下标（如果有多个最大值则为下标中的最小值）。如果两个序列 AAA、BBB 满足：对于任意 1≤l≤r≤n1 \le l \le r \le n1≤l≤r≤n 都满足 RMQ(A,l,r)=RMQ(B,l,r)\mathbf{RMQ}(A, l, r) = \mathbf{RMQ}(B, l, r)RMQ(A,l,r)=RMQ(B,l,r)，那么我们称序列 AAA、BBB RMQ相似。

若已知序列 A={a1,a2,…,an}A = { a_1, a_2, \dots, a_n }A={a1​,a2​,…,an​}，对于序列 B={b1,b2,…,bn}B = {b_1, b_2, \dots, b_n }B={b1​,b2​,…,bn​} 定义序列 BBB 的权重 WBW_BWB​：

WB={∑i=1nbiA,B are RMQ similar0otherwiseW_B =
\begin{cases}
\sum\limits_{i = 1}^{n} b_i & A,B \text{ are RMQ similar} \
0 & \text{otherwise}
\end{cases}WB​=⎩⎪⎨⎪⎧​i=1∑n​bi​0​A,B are RMQ similarotherwise​

现给定序列 AAA，且已知序列 BBB 是由 (0,1)(0,1)(0,1) 间随机实数组成的长度与 AAA 相等的序列。试求 WBW_BWB​ 的期望。期望值的形式是 pq\frac{p}{q}qp​，而你需要输出 pq−1(mod109+7)pq^{-1} \pmod{10^9 + 7}pq−1(mod109+7)。

分析

根据题目定义，如果两个序列 RMQ相似，通俗地讲就是两个序列每个区间内最大值的位置都相同。这令我们联想到笛卡尔树这一数据结构。

我们先简单回顾一下笛卡尔树的特性：

  • 每个节点有两个键值：indexindexindex 和 valuevaluevalue；
  • indexindexindex 满足二叉搜索树性质，valuevaluevalue 满足堆性质。

这里如果我们把 indexindexindex 当作每个元素的下标，valuevaluevalue 当作每个元素的值，那么很容易可以发现两个序列 RMQ 相似 即两个序列的笛卡尔树同构。

我们先对 AAA 建立一棵笛卡尔树。记笛卡尔树上以第 iii 个节点为根节点的子树的大小为 size[i]size[i]size[i]。在这个大小为 size[i]size[i]size[i] 的子树中，对于任意一个节点都有 1size[i]\frac{1}{size[i]}size[i]1​ 的概率成为根节点。那么根据乘法原理，AAA、BBB 两个序列上构造的笛卡尔树同构的概率即：

∏i=1n1size[i]\prod\limits_{i = 1}^{n} \frac{1}{size[i]}i=1∏n​size[i]1​

然后由于 BBB 中每个元素都是均匀地随机在 (0,1)(0, 1)(0,1) 中取值， 所以取值的期望是 12\frac{1}{2}21​。因此 WBW_BWB​ 的期望即：

EWB=∏i=1n1size[i]⋅12+(1−∏i=1n1size[i])⋅0=∏i=1n12⋅size[i]\begin{aligned}
E_{W_B} & = \prod\limits_{i = 1}^{n} \frac{1}{size[i]} \cdot \frac{1}{2} + (1 - \prod\limits_{i = 1}^{n} \frac{1}{size[i]}) \cdot 0 \
& = \prod\limits_{i = 1}^{n} \frac{1}{2 \cdot size[i]}
\end{aligned}EWB​​​=i=1∏n​size[i]1​⋅21​+(1−i=1∏n​size[i]1​)⋅0=i=1∏n​2⋅size[i]1​​
