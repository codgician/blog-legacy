我们先对 aia_iai​ 打个表吧（逃

我们观察这些值，不难发现从第 222 个值开始，111 出现了 111 次，222 出现了 222 次，333 出现了 111 次，444 出现了 333 次…… 好像所有奇数值只会出现 111 次，而能被 222 整除但不能被 444 整除的都出现 222 次，能被 444 整除但不能被 888 整除的都出现 333 次…… 如果我们结合每个数的二进制再观察一番就会发现，除 111 以外每个数 iii 的出现次数 f(i)f(i)f(i) 都满足：

f(i)=log⁡2(lowbit(i))+1f(i) = \log_{2}({\text{lowbit}(i))} + 1f(i)=log2​(lowbit(i))+1

如果你不理解上面这个式子，你大可不必管它。你只需要知道：对于能被 2i2^i2i 整除但不能被 2i+12^{i + 1}2i+1 整除的数 iii，其 f(i)=i+1f(i) = i + 1f(i)=i+1。也就是说，f(i)f(i)f(i) 的值即 iii 中含有质因子 222 的最大次数加 111。

为了处理 111 这一特例，我们不妨把 a1a_1a1​ 扔掉，然后把 aia_iai​ 看作 ai−1a_{i - 1}ai−1​。最后再在答案上加 111。

现在我们不妨把出现了 f(i)f(i)f(i) 次的 iii 都看成一个长度为 f(i)f(i)f(i) 的块。直观地说，就是把原序列这样划分：

[1], [2,2], [3], [4,4,4], [5], [6,6], [7], [8,8,8], …[1],\ [2, 2], \ [3], \ [4, 4, 4], \ [5], \ [6,6], \ [7], \ [8, 8, 8], \ \dots[1], [2,2], [3], [4,4,4], [5], [6,6], [7], [8,8,8], …

那么前 kkk 个数里面应该会有 ttt 个这样完整的块，另外最后可能还有一个不完整的块。我们需要求的 sks_ksk​，也就是前面完整块的和，再加上最后一个不完整块的和（如果有的话）。

前 ttt 个完整块的和：

∑i=1t(i⋅f(i))\sum\limits_{i = 1}^{t} (i \cdot f(i))i=1∑t​(i⋅f(i))

前 ttt 个完整块一共包含数的个数：

∑i=1tf(i)\sum\limits_{i = 1}^{t} f(i)i=1∑t​f(i)

所以剩余数的个数：

k−∑i=1tf(i)k - \sum\limits_{i = 1}^{t} f(i)k−i=1∑t​f(i)

我们知道这个剩余的不完整的块中所有的数都是 t+1t + 1t+1，所以不完整块的和：

(t+1)⋅(k−∑i=1tf(i))(t + 1) \cdot (k - \sum\limits_{i = 1}^{t} f(i))(t+1)⋅(k−i=1∑t​f(i))

加起来便有：

sk=∑i=1t(i⋅f(i))+(t+1)⋅(k−∑i=1tf(i))s_k = \sum\limits_{i = 1}^{t} (i \cdot f(i)) + (t + 1) \cdot (k - \sum_{i = 1}^{t} f(i))sk​=i=1∑t​(i⋅f(i))+(t+1)⋅(k−i=1∑t​f(i))

如何确定 ttt 的值呢？我们可以考虑二分答案把求解转为判定。

再次温习一遍 f(i)f(i)f(i) 的规律：对于能被 2i2^i2i 整除但不能被 2i+12^{i + 1}2i+1 整除的数 iii，有 f(i)=i+1f(i) = i + 1f(i)=i+1。

于是我们不难得出：

∑i=1tf(i)=⌊t20⌋+⌊t21⌋+⌊t22⌋+…\sum\limits_{i = 1}^{t} f(i) = \lfloor \frac{t}{2^0} \rfloor + \lfloor \frac{t}{2^1} \rfloor + \lfloor \frac{t}{2^2} \rfloor + \dotsi=1∑t​f(i)=⌊20t​⌋+⌊21t​⌋+⌊22t​⌋+…

我们不妨把 fff 想象成一堆桶。刚开始时对于所有数都有 f(i)=0f(i) = 0f(i)=0。在上面的式子中，⌊t20⌋\lfloor \frac{t}{2^0} \rfloor⌊20t​⌋ 相当于给所有能被 202^020 数的 fff 都加了 111，⌊t21⌋\lfloor \frac{t}{2^1} \rfloor⌊21t​⌋ 相当于给所有能被 212^121 整除的数的 fff 都加了 111，…… 也就是说，能被 2i2^i2i 整除但不能被 2i+12^{i + 1}2i+1 整除的数的 fff 都被加了 i+1i + 1i+1 次 111。

另外二分时要注意两点：

  • 不可以直接用等比数列求和公式，因为每一项都向下取整了；
  • 关于答案范围：通过前面的表可以观察到 ttt 的值一般在 k2\frac{k}{2}2k​ 左右，且很多时候略大于 k2\frac{k}{2}2k​，故为了卡进时限可以将二分范围确立为 [k2−1,k2+50][\frac{k}{2} - 1, \frac{k}{2} + 50][2k​−1,2k​+50]。

至于前 ttt 个完整块的求和，我们不难发现如下规律：

  • 出现次数为 111 的数：1, 3, 5, 7, 9, …1, \ 3, \ 5, \ 7, \ 9, \ \dots1, 3, 5, 7, 9, … 是个首项为 202^020，公差为 212^121 的等差数列。
  • 出现次数为 222 的数：2, 6, 10, 14, 18, …2, \ 6, \ 10, \ 14, \ 18, \ \dots2, 6, 10, 14, 18, … 进一步，可化为：21×(1, 3, 5, 7, 9, … )2^1 \times (1, \ 3, \ 5, \ 7, \ 9, \ \dots)21×(1, 3, 5, 7, 9, …)
  • 出现次数为 333 的数：4, 12, 20, 28, 36, …4, \ 12, \ 20, \ 28, \ 36, \ \dots4, 12, 20, 28, 36, … 进一步，可化为：22×(1, 3, 5, 7, 9, … )2^2 \times (1, \ 3, \ 5, \ 7, \ 9, \ \dots)22×(1, 3, 5, 7, 9, …)
  • …\dots…
  • 出现次数为 iii 的数：2i−1×(1, 3, 5, 7, 9, … )2^{i - 1} \times (1, \ 3, \ 5, \ 7, \ 9, \ \dots)2i−1×(1, 3, 5, 7, 9, …)

所以我们用等差数列求和公式即可得解。
