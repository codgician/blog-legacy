题面

比特城里有 nnn 个路口，mmm 条单向街道，第 iii 条街道的长度为 wiw_iwi​。最近小Q计划坚持长走 qqq 天：在第 iii 天，小Q计划从 sis_isi​ 路口出发，通过至少 kik_iki​ 条街道并最终到达 tit_iti​ 路口。

虽然小Q看起来很爱运动但其实很懒，所以他希望能使得他每天所走的路程最少。请编写程序计算小Q每一天最少需要走的距离。

数据范围：

TTT 组测试数据，1≤T≤101 \le T \le 101≤T≤10

2≤n≤502 \le n \le 502≤n≤50

1≤m,wi,ki≤100001 \le m, w_i, k_i \le 100001≤m,wi​,ki​≤10000

分析

记 G[i][j]G[i][j]G[i][j] 代表一步从 iii 到 jjj 的距离（需要注意的是当 i=ji = ji=j 时 G[i][j]=+∞G[i][j] = +\inftyG[i][j]=+∞，因为图中没有自环所以一步是不可能还待在出发点的）。另外，记 f[t][i][j]f[t][i][j]f[t][i][j] 代表从 iii 出发恰好经过 kkk 条边到达 jjj 的最短路。显然：

f[t][i][j]=f[t−1][i][k]+G[k][j]f[t][i][j] = f[t - 1][i][k] + G[k][j]f[t][i][j]=f[t−1][i][k]+G[k][j]

另外，对于 fff 本身，也有：

f[t][i][j]=min⁡1≤x<t{f[x][i][k]+f[t−x][k][j]}f[t][i][j] = \min_{1 \le x < t}{f[x][i][k] + f[t - x][k][j] }f[t][i][j]=1≤x<tmin​{f[x][i][k]+f[t−x][k][j]}

我们不妨把这一合并过程看作类似矩阵乘法的过程，那么就有：

f[t]=Gtf[t] = G^tf[t]=Gt

而每执行这样一次乘法的过程复杂度就高达 O(n3)\mathcal{O}(n^3)O(n3)，如果我们要朴素地预处理出 fff 的话复杂度便高达 O(kn3)\mathcal{O}(kn^3)O(kn3)，显然是不可接受的。即使我们尝试用矩阵快速幂优化，那么单次查询复杂度便高达 O(n3log⁡k)\mathcal{O}(n^3\log{k})O(n3logk)，乘上 mmm 后显然也是不可接受的。

于是我们只好求助…… 万能的分块（逃

我们可以取块长 L=10000=100L = \sqrt{10000} = 100L=10000​=100。记 A=⌊k100⌋A = \lfloor \frac{k}{100} \rfloorA=⌊100k​⌋，B=kmod  100B = k \mod 100B=kmod100。

然后我们把每次询问的过程分成两步：

  • 从 sss 出发恰好经过 L⋅AL \cdot AL⋅A 条边到达某个点 uuu；
  • 从 uuu 出发至少经过 BBB 条边到达 ttt。

这样一来我们可以考虑开两个数组 fAf_AfA​ 和 fBf_BfB​。其中 fA[t][i][j]f_A[t][i][j]fA​[t][i][j] 表示从 iii 到 jjj 恰好经过 L⋅tL \cdot tL⋅t 条边到达 jjj 的最短路，而 fB[t][i][j]f_B[t][i][j]fB​[t][i][j] 则表示从 iii 到 jjj 至少经过 ttt 条边到达 jjj 的最短路。那么对于每一次询问，其答案实际上就是：

ans=min⁡{fA[A][s][u]+fB[B][u][t]}ans = \min { f_A[A][s][u] + f_B[B][u][t] }ans=min{fA​[A][s][u]+fB​[B][u][t]}
只需要枚举 uuu 就可以了，所以每一次询问我们可以在 O(n)\mathcal{O}(n)O(n) 的时间复杂度内解决。

接下来谈谈预处理。首先，fAf_AfA​ 可以通过如下方式得到：

fA[t]=(GL)tf_A[t] = (G{L})tfA​[t]=(GL)t

但是对于 fBf_BfB​，由于它表示的是至少经过 ttt 条边，我们还需要另跑一次 Floyd，得到最短路矩阵 dis[i][j]dis[i][j]dis[i][j]（表示从 iii 到 jjj 的最短路），然后我们再把 GtG^tGt 和 dis[i][j]dis[i][j]dis[i][j] 按照之前说的方式合并，就得到了经过至少 ttt 条边从 iii 到 jjj 的最短路。

预处理复杂度就被压到了 O(n3k)\mathcal{O}(n^3\sqrt{k})O(n3k​)。这样一来，总的复杂度就是 O(n3k+qn)\mathcal{O}(n^3\sqrt{k} + qn)O(n3k​+qn)。
