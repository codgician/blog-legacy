题面

给定 n,m,kn, m, kn,m,k，求不定方程：

∑i=1mxi=k (xi∈Z,0≤xi<n)\sum\limits_{i = 1}^{m} x_i = k \ (x_i \in Z, 0 \le x_i < n)i=1∑m​xi​=k (xi​∈Z,0≤xi​<n)
的解的个数。

数据范围：

1≤n,m≤1051 \le n, m \le 10^51≤n,m≤105

0≤k≤1050 \le k \le 10^50≤k≤105

所有数据 n,m,kn, m, kn,m,k 之和不会超过 5×1065 \times 10^65×106。

分析

首先，如果题目没有给出 xi<nx_i < nxi​<n 这一限制，我们可以用隔板法（参考 HDUOJ 3037 - Saving Beans）求出解的种数：

(k+m−1m−1)\binom{k + m - 1}{m - 1}(m−1k+m−1​)

但是现在我们有 xi<nx_i < nxi​<n 这一限制条件，因此我们需要把存在 xi≥nx_i \ge nxi​≥n 的解剔除掉。我们考虑使用容斥原理。

为了方便叙述，我们以 m=3m = 3m=3 的情况为例，画出如下韦恩图：

我们所要求的就是 x1,x2,x3<nx_1, x_2, x_3 < nx1​,x2​,x3​<n 的部分（即上图中的 S0S_0S0​）。

我们先用容斥原理求出存在 xix_ixi​ 不满足上限的全集的大小（即 ∣S1∪S2∪S3∣|S_1 \cup S_2 \cup S_3|∣S1​∪S2​∪S3​∣）：

∣⋃i=13Si∣=(∣S1∣+∣S2∣+∣S3∣)−(∣S1∪S2∣+∣S1∪S3∣+∣S2∪S3∣)+∣S1∪S2∪S3∣=∑i=13∣Si∣−∑1≤i<j≤3∣Si∪Sj∣+∑1≤i<j<k≤3∣Si∪Sj∪Sk∣\begin{aligned}
|\bigcup\limits_{i = 1}^{3} S_i| &= (|S_1| + |S_2| + |S_3|) - (|S_1 \cup S_2| + |S_1 \cup S_3| + |S_2 \cup S_3|) + |S_1 \cup S_2 \cup S_3| \
& = \sum\limits_{i = 1}^{3}|S_i| - \sum\limits_{1 \le i < j \le 3}|S_i \cup S_j| + \sum\limits_{1 \le i < j < k \le 3} |S_i \cup S_j \cup S_k|
\end{aligned}∣i=1⋃3​Si​∣​=(∣S1​∣+∣S2​∣+∣S3​∣)−(∣S1​∪S2​∣+∣S1​∪S3​∣+∣S2​∪S3​∣)+∣S1​∪S2​∪S3​∣=i=1∑3​∣Si​∣−1≤i<j≤3∑​∣Si​∪Sj​∣+1≤i<j<k≤3∑​∣Si​∪Sj​∪Sk​∣​

那么我们所要求的答案就是所有方案数减去存在 xix_ixi​ 不满足上限的方案数：

∣S0∣=∣S0∪S1∪S2∪S3∣−∣S1∪S2∪S3∣|S_0| = |S_0 \cup S_1 \cup S_2 \cup S_3| - |S_1 \cup S_2 \cup S_3|∣S0​∣=∣S0​∪S1​∪S2​∪S3​∣−∣S1​∪S2​∪S3​∣

我们可以轻松从 m=3m = 3m=3 推广到 mmm 为任意值的情况：

∣S0∣=∣⋃i=0mSi∣−∣⋃i=1mSi∣=(k+m−1m−1)−[∑i=1m∣Si∣−∑1≤i<j≤m∣Si∪Sj∣+⋯+(−1)m+1∣S1∪⋯∪Sm∣]\begin{aligned}
|S_0| & = |\bigcup\limits_{i = 0}^{m} S_i| - |\bigcup\limits_{i = 1}^{m} S_i| \
& = \binom{k + m - 1}{m - 1} - \left[\sum\limits_{i = 1}^{m} |S_i| - \sum\limits_{1 \le i < j \le m} |S_i \cup S_j| + \dots + (-1)^{m + 1}|S_1 \cup \dots \cup S_m| \right]
\end{aligned}∣S0​∣​=∣i=0⋃m​Si​∣−∣i=1⋃m​Si​∣=(m−1k+m−1​)−[i=1∑m​∣Si​∣−1≤i<j≤m∑​∣Si​∪Sj​∣+⋯+(−1)m+1∣S1​∪⋯∪Sm​∣]​

接下来我们需要使用组合数学的相关知识把上述公式中的每一项表示出来。

记 F(p)F§F(p) 代表从 mmm 个 xix_ixi​ 中选取 ppp 个来违反上界限制的种数，那么上述公式可改写为：

∣S0∣=(k+m−1m−1)−[F(1)−F(2)+⋯+(−1)m+1F(m)]=(k+m−1m−1)+∑i=1m(−1)i⋅F(i)\begin{aligned}
|S_0| & = \binom{k + m - 1}{m - 1} - \left[ F(1) - F(2) + \dots + (-1)^{m + 1} F(m) \right] \
& = \binom{k + m - 1}{m - 1} + \sum\limits_{i = 1}^{m} (-1)^i \cdot F(i)
\end{aligned}∣S0​∣​=(m−1k+m−1​)−[F(1)−F(2)+⋯+(−1)m+1F(m)]=(m−1k+m−1​)+i=1∑m​(−1)i⋅F(i)​

我们来思考如何表示出 F(i)F(i)F(i)。记选出违反上界限制的 ppp 个未知量为 x1,x2,…xpx_1, x_2, \dots x_px1​,x2​,…xp​，则：

x1+x2+⋯+xm=k⇒(x1−n)+(x2−n)+⋯+(xp−n)+xp+1+⋯+xm=k−p⋅n\begin{aligned}
x_1 + x_2 + \dots + x_m & = k\
\Rightarrow (x_1 - n) + (x_2 - n) + \dots + (x_p - n) + x_{p + 1} + \dots + x_m & = k - p \cdot n
\end{aligned}x1​+x2​+⋯+xm​⇒(x1​−n)+(x2​−n)+⋯+(xp​−n)+xp+1​+⋯+xm​​=k=k−p⋅n​

由此一来我们选出的 ppp 个不违反上界限制的变量都保证满足上界限制了（当然未选出的变量仍有可能违反上界限制）。套用本文开头所述的公式得解的组数为：(k−p⋅n+m−1m−1)\binom{k - p \cdot n + m - 1}{m - 1}(m−1k−p⋅n+m−1​)，而在 mmm 个 xix_ixi​ 中任选 ppp 个的方案为 (mp)\binom{m}{p}(pm​)， 由乘法原理得：

F(p)=(mp)⋅(k−p⋅n+m−1m−1)F§ = \binom{m}{p} \cdot \binom{k - p \cdot n + m - 1}{m - 1}F(p)=(pm​)⋅(m−1k−p⋅n+m−1​)

我们将其带入上面我们得到的公式就可以求得解了。
