简介

最近经常演人/被人演，所以是时候学一学反演了。

莫比乌斯函数

定义

对于正整数 nnn，若对其分解质因数有 n=p1c1p2c2…pkck (ci>0)n = p_1^{c_1} p_2^{c_2} \dots p_k^{c_k} \ (c_i > 0)n=p1c1​​p2c2​​…pkck​​ (ci​>0)，则：

μ(n)={1n=10∃ci>1(−1)kotherwise\mu(n) =
\begin{cases}
1 & n = 1 \
0 & \exists c_i > 1 \
(-1)^k & \text{otherwise}
\end{cases}μ(n)=⎩⎪⎪⎨⎪⎪⎧​10(−1)k​n=1∃ci​>1otherwise​

通俗地来讲，当 n=1n = 1n=1 时，μ(n)=1\mu(n) = 1μ(n)=1；当 nnn 的因子中存在完全平方数时，μ(n)=0\mu(n) = 0μ(n)=0。对于剩余情况，也就是对 nnn 质因数分解可得到 n=p1p2…pkn = p_1p_2 \dots p_kn=p1​p2​…pk​ 的情况，μ(n)=(−1)k\mu(n) = (-1)^kμ(n)=(−1)k，即 μ(n)\mu(n)μ(n) 的值取决于 nnn 不同质因数的个数。

性质

∑d∣nμ(d)={1n=10otherwise\sum\limits_{d \mid n} \mu(d) =
\begin{cases}
1 & n = 1\
0 & \text{otherwise} \
\end{cases}d∣n∑​μ(d)={10​n=1otherwise​

下面给出简要证明：

  1. n=1n = 1n=1 时，显然有 ∑d∣nμ(d)=1\sum\limits_{d \mid n} \mu(d) = 1d∣n∑​μ(d)=1；
  2. n≠1n \neq 1n​=1 时，有 k>0k > 0k>0。记 n=p1c1p2c2…pkck (ci>0)n = p_1{c_1}p_2{c_2} \dots p_k^{c_k} \ (c_i > 0)n=p1c1​​p2c2​​…pkck​​ (ci​>0)。对于 d∣nd \mid nd∣n，μ(d)≠0\mu(d) \neq 0μ(d)​=0 当且仅当 ddd 中不存在完全平方数因子。显然，具有 iii 个质因数的 ddd 有 (ki)\binom{k}{i}(ik​) 个。因此，∑d∣nμ(d)=∑i=0k(−1)k(ki)\sum\limits_{d \mid n} \mu(d) = \sum\limits_{i = 0}^{k} (-1)^k\binom{k}{i}d∣n∑​μ(d)=i=0∑k​(−1)k(ik​)。我们不难观察发现，这个式子就是一个二项式展开，即 (1−1)k(1 - 1)^k(1−1)k，故其值为 000。

莫比乌斯反演

形式 #1

内容

定义 F(n),f(n)F(n), f(n)F(n),f(n) 均为非负整数集合上的两个函数，则有：

F(n)=∑d∣nf(d)⇔f(n)=∑d∣nμ(d)F(nd)F(n) = \sum\limits_{d \mid n}f(d) \Leftrightarrow f(n) = \sum\limits_{d \mid n} \mu(d)F(\frac{n}{d})F(n)=d∣n∑​f(d)⇔f(n)=d∣n∑​μ(d)F(dn​)

简而言之，利用莫比乌斯反演，只要我们知道 F(n)F(n)F(n) 或是 f(n)f(n)f(n) 中一者的定义，就可以推知另一者的具体定义。

证明

下面我们给出莫比乌斯反演的一种简要证明：

∑d∣nμ(d)F(nd)=∑d∣nμ(d)∑k∣ndf(k)=∑k∣nf(k)∑d∣nkμ(d)\begin{aligned}
\sum\limits_{d \mid n} \mu(d)F(\frac{n}{d}) = & \sum\limits_{d \mid n} \mu(d) \sum\limits_{k \mid \frac{n}{d}} f(k) \
= & \sum\limits_{k \mid n} f(k) \sum\limits_{d \mid \frac{n}{k}} \mu(d) \
\end{aligned}d∣n∑​μ(d)F(dn​)==​d∣n∑​μ(d)k∣dn​∑​f(k)k∣n∑​f(k)d∣kn​∑​μ(d)​

由前文提到的性质，当且仅当 nk=1\frac{n}{k} = 1kn​=1 时 ∑d∣nkμ(d)=1\sum\limits_{d \mid \frac{n}{k}} \mu(d) = 1d∣kn​∑​μ(d)=1；否则该式值为 000。所以：

∑d∣nμ(d)F(nd)=f(k)\begin{aligned}
\sum\limits_{d \mid n} \mu(d)F(\frac{n}{d}) = f(k)
\end{aligned}d∣n∑​μ(d)F(dn​)=f(k)​

得证。

形式 #2

内容

定义 F(n),f(n)F(n), f(n)F(n),f(n) 均为非负整数集合上的两个函数，则有：

F(n)=∑n∣df(d)⇔f(n)=∑n∣dμ(dn)F(d)F(n) = \sum\limits_{n \mid d}f(d) \Leftrightarrow f(n) = \sum\limits_{n \mid d} \mu(\frac{d}{n})F(d)F(n)=n∣d∑​f(d)⇔f(n)=n∣d∑​μ(nd​)F(d)

注意这一形式与上一形式最大的不同点，即求和条件由 d∣nd \mid nd∣n 变为 n∣dn \mid dn∣d。

证明

令 t=dnt = \frac{d}{n}t=nd​，有：

∑n∣dμ(dn)F(d)=∑t=1+∞μ(t)F(nt)=∑t=1+∞μ(t)∑nt∣kf(k)=∑n∣kf(k)∑t∣knμ(t)\begin{aligned}
\sum\limits_{n \mid d} \mu(\frac{d}{n})F(d) = & \sum\limits_{t = 1}^{+\infty}\mu(t)F(nt) \
= & \sum\limits_{t = 1}^{+\infty}\mu(t)\sum\limits_{nt \mid k}f(k) \
= & \sum\limits_{n \mid k}f(k)\sum\limits_{t \mid \frac{k}{n}}\mu(t)
\end{aligned}n∣d∑​μ(nd​)F(d)===​t=1∑+∞​μ(t)F(nt)t=1∑+∞​μ(t)nt∣k∑​f(k)n∣k∑​f(k)t∣nk​∑​μ(t)​

由前文提到的性质，当且仅当 kn=1\frac{k}{n} = 1nk​=1 时 ∑t∣knμ(t)=1\sum\limits_{t \mid \frac{k}{n}}\mu(t) = 1t∣nk​∑​μ(t)=1，否则该式值为 000。所以：

∑n∣dμ(dn)F(d)=f(n)\sum\limits_{n \mid d} \mu(\frac{d}{n})F(d) = f(n)n∣d∑​μ(nd​)F(d)=f(n)

应用

当 F(n)F(n)F(n) 和 f(n)f(n)f(n) 中一者容易求得，而另一着不易求得时，我们可以借助莫比乌斯反演来求得不易求得的函数。

与欧拉函数联系

这里我们尝试应用一下莫比乌斯反演来推导莫比乌斯函数 μ(n)\mu(n)μ(n) 与欧拉函数 φ(n)\varphi(n)φ(n) 间的关系。

首先欧拉函数有一个性质：

∑d∣nφ(d)=n\sum\limits_{d \mid n} \varphi(d) = nd∣n∑​φ(d)=n

那么我们不妨令 F(n)=∑d∣nφ(d)=nF(n) = \sum\limits_{d \mid n} \varphi(d) = nF(n)=d∣n∑​φ(d)=n，运用莫比乌斯反演可得：

φ(n)=∑d∣nμ(d)F(nd)=∑d∣nμ(d)⋅nd\begin{aligned}
\varphi(n) = & \sum\limits_{d \mid n} \mu(d)F(\frac{n}{d}) \
= & \sum\limits_{d \mid n} \mu(d) \cdot \frac{n}{d}
\end{aligned}φ(n)==​d∣n∑​μ(d)F(dn​)d∣n∑​μ(d)⋅dn​​

整理后可得：

∑d∣nμ(d)d=φ(n)n\sum\limits_{d \mid n} \frac{\mu(d)}{d} = \frac{\varphi(n)}{n}d∣n∑​dμ(d)​=nφ(n)​

注：其实这个结论也可以利用 φ(n)\varphi(n)φ(n) 的计算公式加上容斥原理得出，而在其过程中，实际上容斥系数就是 μ(d)\mu(d)μ(d)。

与最大公约数联系

给定 n,mn, mn,m，求满足 1≤i≤n, 1≤j≤m, gcd⁡(i,j)=11 \le i \le n, \ 1 \le j \le m, \ \gcd(i, j) = 11≤i≤n, 1≤j≤m, gcd(i,j)=1 的数对 ⟨i,j⟩\langle i, j \rangle⟨i,j⟩ 个数（其中 ⟨a,b⟩\langle a, b \rangle⟨a,b⟩ 和 ⟨b,a⟩\langle b, a \rangle⟨b,a⟩ 算两个不同的数对）。

换言之，即求：

∑i=1n∑j=1m[gcd⁡(i,j)=1]\sum\limits_{i = 1}^{n}\sum\limits_{j = 1}^{m} [\gcd(i, j) = 1]i=1∑n​j=1∑m​[gcd(i,j)=1]

暴力的做法显然复杂度是 O(n2)\mathcal{O}(n^2)O(n2) 级别的。

我们可以考虑利用莫比乌斯函数的性质：

∑i=1n∑j=1m[gcd⁡(i,j)=1]=∑i=1n∑j=1m∑d∣gcd⁡(i,j)μ(d)=∑d=1min⁡(n,m)μ(d)∑i=1n∑j=1m[d∣gcd⁡(i,j)]=∑d=1min⁡(n,m)μ(d)⌊nd⌋⌊md⌋\begin{aligned}
\sum\limits_{i = 1}^{n}\sum\limits_{j = 1}^{m} [\gcd(i, j) = 1] = & \sum\limits_{i = 1}^{n}\sum\limits_{j = 1}^{m}\sum\limits_{d \mid \gcd(i, j)}\mu(d) \
= & \sum\limits_{d = 1}^{\min(n, m)}\mu(d) \sum\limits_{i = 1}^{n}\sum\limits_{j = 1}^{m} [d \mid \gcd(i, j)] \
= & \sum\limits_{d = 1}^{\min(n, m)} \mu(d)\left\lfloor \frac{n}{d} \right\rfloor \left\lfloor \frac{m}{d} \right\rfloor
\end{aligned}i=1∑n​j=1∑m​[gcd(i,j)=1]===​i=1∑n​j=1∑m​d∣gcd(i,j)∑​μ(d)d=1∑min(n,m)​μ(d)i=1∑n​j=1∑m​[d∣gcd(i,j)]d=1∑min(n,m)​μ(d)⌊dn​⌋⌊dm​⌋​

我们可以用线性筛以 O(N)\mathcal{O}(N)O(N) 的复杂度预处理出 μ(n)\mu(n)μ(n) 的前缀和从而能够 O(1)\mathcal{O}(1)O(1) 回答 μ(n)\mu(n)μ(n) 的区间和。而面对询问时，我们可以用 O(N)\mathcal{O}(\sqrt{N})O(N​) 的复杂度进行整除分块并进行求解。

那么如果我们要求的是 gcd⁡(i,j)=k\gcd(i, j) = kgcd(i,j)=k 怎么办？我们仅需做如下变换：

∑i=1n∑j=1m[gcd⁡(i,j)=k]=∑i=1⌊nk⌋∑j=1⌊mk⌋[gcd⁡(i,j)=1]\sum\limits_{i = 1}^{n}\sum\limits_{j = 1}^{m} [\gcd(i, j) = k] = \sum\limits_{i = 1}^{\left\lfloor \frac{n}{k} \right\rfloor}\sum\limits_{j = 1}^{\left\lfloor \frac{m}{k} \right\rfloor} [\gcd(i, j) = 1]i=1∑n​j=1∑m​[gcd(i,j)=k]=i=1∑⌊kn​⌋​j=1∑⌊km​⌋​[gcd(i,j)=1]

接下来就跟之前的做法一样咯，所以我们能得到：

Ans=∑d=1min⁡(⌊nk⌋,⌊mk⌋)μ(d)⌊nkd⌋⌊mkd⌋Ans = \sum\limits_{d = 1}^{\min(\left\lfloor \frac{n}{k} \right\rfloor, \left\lfloor \frac{m}{k} \right\rfloor)} \mu(d) \left\lfloor \frac{n}{kd} \right\rfloor \left\lfloor \frac{m}{kd} \right\rfloorAns=d=1∑min(⌊kn​⌋,⌊km​⌋)​μ(d)⌊kdn​⌋⌊kdm​⌋

当然，我们也可以向莫比乌斯反演的的方向进行考虑。

我们不妨记 f(k)f(k)f(k) 代表满足 gcd⁡(i,j)=k\gcd(i, j) = kgcd(i,j)=k 的 ⟨i,j⟩\langle i, j \rangle⟨i,j⟩ 对个数。那么 F(k)=∑k∣df(d)F(k) = \sum\limits_{k \mid d}f(d)F(k)=k∣d∑​f(d) 的意义即为满足
k∣gcd⁡(i,j)k \mid \gcd(i, j)k∣gcd(i,j) 的 ⟨i,j⟩\langle i, j \rangle⟨i,j⟩ 对数。而求满足 1≤i≤n, 1≤j≤m1 \le i \le n, \ 1 \le j \le m1≤i≤n, 1≤j≤m 范围内 k∣gcd⁡(i,j)k \mid \gcd(i, j)k∣gcd(i,j) 这一条件的对数显然等价于 1≤i≤⌊nk⌋, 1≤j≤⌊mk⌋1 \le i \le \left\lfloor \frac{n}{k} \right\rfloor, \ 1 \le j \le \left\lfloor \frac{m}{k} \right\rfloor1≤i≤⌊kn​⌋, 1≤j≤⌊km​⌋ 范围内 1∣gcd⁡(i,j)1 \mid \gcd(i, j)1∣gcd(i,j) 的对数（显然在此范围内的所有数对都满足这一条件）。由此我们可以很容易得到 F(k)F(k)F(k) 的具体定义：

F(k)=⌊nk⌋⌊mk⌋F(k) = \left\lfloor \frac{n}{k} \right\rfloor \left\lfloor \frac{m}{k} \right\rfloorF(k)=⌊kn​⌋⌊km​⌋

接下来我们就可以利用莫比乌斯反演直接得到 f(k)f(k)f(k) 的具体定义了：

f(k)=∑k∣dμ(dk)F(d)=∑k∣dμ(dk)⌊nd⌋⌊md⌋\begin{aligned}
f(k) = & \sum\limits_{k \mid d} \mu(\frac{d}{k})F(d) \
= & \sum\limits_{k \mid d} \mu(\frac{d}{k}) \left\lfloor \frac{n}{d} \right\rfloor \left\lfloor \frac{m}{d} \right\rfloor \
\end{aligned}f(k)==​k∣d∑​μ(kd​)F(d)k∣d∑​μ(kd​)⌊dn​⌋⌊dm​⌋​

不妨令 t=dkt = \frac{d}{k}t=kd​，我们便得到了跟之前一样的结果：

f(k)=∑t=1min⁡(⌊nk⌋,⌊mk⌋)μ(t)⌊ntk⌋⌊mtk⌋f(k) = \sum\limits_{t = 1}^{\min(\left\lfloor \frac{n}{k} \right\rfloor, \left\lfloor \frac{m}{k} \right\rfloor)} \mu(t) \left\lfloor \frac{n}{tk} \right\rfloor \left\lfloor \frac{m}{tk} \right\rfloorf(k)=t=1∑min(⌊kn​⌋,⌊km​⌋)​μ(t)⌊tkn​⌋⌊tkm​⌋
